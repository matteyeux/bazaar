# bazaar

### Usage

```
poetry run bazaar_cli/main.py                                                                               
Usage: main.py [OPTIONS] COMMAND [ARGS]...

Options:
  -v, --version  print version
  -d, --docs     access documentation
  --help         Show this message and exit.

Commands:
  download  Download samples.
  info      Sample info.
  list      List samples.
```
